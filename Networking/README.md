# 🌐 Networking Labs

## Overview
This section contains my hands-on networking labs, configurations, troubleshooting exercises, and network design projects.

The goal of this directory is to strengthen practical networking skills through continuous lab work and scenario-based implementations using Cisco technologies and networking tools.

Topics range from foundational CCNA concepts to more advanced routing, switching, and troubleshooting scenarios.

---

# 📁 Structure

## basic-labs/
Contains foundational networking labs focused on core concepts and configurations.

Examples:
- Basic router and switch configuration
- VLAN configuration
- Static routing
- Basic OSPF
- Access Control Lists (ACLs)
- DHCP and NAT fundamentals

These labs focus on building strong networking fundamentals and command-line familiarity.

---

## advanced-labs/
Contains more complex networking scenarios involving:
- Multi-router topologies
- Advanced troubleshooting
- Redundancy concepts
- Network optimization
- Scenario-based configurations
- Simulated enterprise environments

These labs emphasize analytical thinking, troubleshooting methodology, and real-world network behavior.

---

## projects/
Contains larger networking implementations and end-to-end design projects.

Projects may include:
- Small office network simulations
- Enterprise-style network design
- Segmentation and security implementations
- Documentation-focused deployments
- Infrastructure planning and validation
